# Privacy Policy

Effective date: May 26, 2026

## Overview

ReelPlus is a browser extension that adds playback controls for Instagram Reels, including play/pause, replay, mute/unmute, volume controls, a seek slider, keyboard shortcuts, and a draggable control panel.

ReelPlus does not collect, sell, transmit, or share personal data.

## Data Collection

ReelPlus does not collect any personal information.

The extension does not collect:

- Names
- Email addresses
- Account information
- Instagram login details
- Messages
- Photos or videos
- Browsing history
- Analytics data
- Advertising identifiers

## Local Storage

ReelPlus uses Chrome local storage only to save extension preferences on the user's device.

This may include:

- The floating panel position
- Interface preferences
- Playback-related preferences, if added in future versions

This data stays on the user's device and is not sent to any external server.

## Network Requests

ReelPlus does not make external network requests.

## Third-Party Sharing

ReelPlus does not sell, rent, transfer, or share user data with third parties.

## Permissions

ReelPlus requests the `storage` permission so it can remember local preferences such as the position of the floating control panel.

## Website Access

ReelPlus runs on Instagram pages so it can find and control video elements on the page. The extension only interacts with video playback controls and does not collect page content or user account data.

## Limited Use

ReelPlus uses extension permissions only to provide its single purpose: adding local playback controls for Instagram Reels.

User data is not transferred, sold, or used for advertising, analytics, profiling, or unrelated purposes.

## Affiliation

ReelPlus is an independent project and is not affiliated with Instagram, Meta, or any related company.

## Contact

For questions or issues, please open an issue on the project's GitHub repository.
